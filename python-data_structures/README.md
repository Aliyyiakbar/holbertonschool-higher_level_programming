# Python - Data Structures: Lists, Tuples

This directory contains my solutions for the Holberton School project
**"Python - Data Structures: Lists, Tuples"**.

The goal of this project is to practice Python lists, tuples, sequences,
indexing, replacing values, copying lists, tuple operations, and basic matrix
handling.

## Requirements

- OS: Ubuntu 20.04 LTS
- Interpreter: `python3` (version 3.8.5)
- Style: `pycodestyle` (version 2.7.*)
- All files must be executable
- All files must end with a new line
- The first line of all files must be exactly `#!/usr/bin/python3`
- A `README.md` file at the root of this project folder is mandatory
- File length will be tested using `wc`
- No modules may be imported unless the task allows it

## Files

- `0-print_list_integer.py`  
  Prints all integers of a list, one number per line.

- `1-element_at.py`  
  Retrieves an element from a list safely by index.

- `2-replace_in_list.py`  
  Replaces an element of a list at a specific position.

- `3-print_reversed_list_integer.py`  
  Prints all integers of a list in reverse order.

- `4-new_in_list.py`  
  Replaces an element in a copied list without modifying the original list.

- `5-no_c.py`  
  Removes all characters `c` and `C` from a string.

- `6-print_matrix_integer.py`  
  Prints a matrix of integers with formatted output.

- `7-add_tuple.py`  
  Adds two tuples and returns a tuple with the sum of their first two values.

- `8-multiple_returns.py`  
  Returns the length of a string and its first character.

- `9-max_integer.py`  
  Finds the biggest integer in a list.

- `10-divisible_by_2.py`  
  Returns a list showing whether each integer is divisible by 2.

- `11-delete_at.py`  
  Deletes an item at a specific position in a list.

- `12-switch.py`  
  Switches the values of two variables.

## Usage

Example:

```bash
chmod +x 0-print_list_integer.py
./0-main.py
```
