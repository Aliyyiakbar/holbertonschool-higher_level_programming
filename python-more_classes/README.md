# Python - More Classes and Objects

This directory contains my solutions for the Holberton School project
**"Python - More Classes and Objects"**.

The goal of this project is to continue practicing object-oriented programming
with rectangles, special methods, string representations, class attributes,
static methods, class methods, destructors, and the N queens problem.

## Requirements

- OS: Ubuntu 20.04 LTS
- Interpreter: `python3` (version 3.8.5)
- Style: `pycodestyle` (version 2.7.*)
- All files must be executable
- All files must end with a new line
- The first line of all files must be exactly `#!/usr/bin/python3`
- A `README.md` file at the root of this project folder is mandatory
- File length will be tested using `wc`
- No modules may be imported unless the task allows it

## Files

- `0-rectangle.py`  
  Defines an empty `Rectangle` class.

- `1-rectangle.py`  
  Defines a `Rectangle` class with private width and height attributes.

- `2-rectangle.py`  
  Adds `area` and `perimeter` methods.

- `3-rectangle.py`  
  Adds a string representation that prints the rectangle using `#`.

- `4-rectangle.py`  
  Adds an official representation that can recreate the rectangle with `eval`.

- `5-rectangle.py`  
  Prints a message when a rectangle instance is deleted.

- `6-rectangle.py`  
  Tracks the number of rectangle instances.

- `7-rectangle.py`  
  Adds a customizable print symbol for rectangle representation.

- `8-rectangle.py`  
  Compares two rectangles by area using a static method.

- `9-rectangle.py`  
  Creates a square as a special rectangle using a class method.

- `100-rectangle.py`  
  Explores class and instance attributes in the `Rectangle` class.

- `101-nqueens.py`  
  Solves the N queens problem and prints all valid board solutions.

## Usage

Example:

```bash
chmod +x 0-rectangle.py
./0-main.py
```
