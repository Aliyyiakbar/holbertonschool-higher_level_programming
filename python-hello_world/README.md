# Python - Hello, World

This directory contains my solutions for the Holberton School project
**"Python - Hello, World"**.

The goal of this project is to practice the basics of Python programming,
including using the Python interpreter, printing text and variables, working
with strings, indexing and slicing, and following the `pycodestyle` style guide.

## Requirements

- OS: Ubuntu 20.04 LTS
- Interpreter: `python3` (version 3.8.*)
- Style: `pycodestyle` (version 2.7.*)
- All files must be executable
- All files must end with a new line
- The first line of all files must be exactly `#!/usr/bin/python3`
- A `README.md` file at the root of the repository is mandatory
- A `README.md` file at the root of this project folder is mandatory
- File length will be tested using `wc`

## Files

- `2-print.py`  
  Prints exactly `"Programming is like building a multilingual puzzle`.

- `3-print_number.py`  
  Prints an integer stored in a variable, followed by `Battery street`.

- `4-print_float.py`  
  Prints a float with a precision of 2 digits.

- `5-print_string.py`  
  Prints a string three times, followed by its first 9 characters.

- `6-concat.py`  
  Prints `Welcome to Holberton School!` using string concatenation.

- `7-edges.py`  
  Demonstrates indexing and slicing by extracting specific parts of a word.

- `8-concat_edges.py`  
  Creates a new sentence from parts of the word `object-oriented programming`.

- `9-easter_egg.py`  
  Prints “The Zen of Python” by Tim Peters.

## Usage

Example:

```bash
chmod +x 2-print.py
./2-print.py
```
