# Holberton School - Higher Level Programming

This repository contains my Python programming work for Holberton School.
It covers the fundamentals of higher-level programming: printing, strings,
control flow, loops, functions, modules, data structures, exceptions,
test-driven development, and object-oriented programming.

## Requirements

- OS: Ubuntu 20.04 LTS / Ubuntu 22.04 LTS depending on the project
- Interpreter: `python3`
- Style: `pycodestyle` (version 2.7.*)
- All files end with a new line
- All Python files are executable
- The first line of Python files is `#!/usr/bin/python3`

Each project directory has its own `README.md` with task details and file descriptions.

## Projects

- `python-hello_world` — first steps in Python, printing, strings, indexing, and slicing
- `python-if_else_loops_functions` — conditionals, loops, functions, scope, and arithmetic operations
- `python-import_modules` — imports, modules, command-line arguments, and calculator scripts
- `python-data_structures` — lists, tuples, indexing, matrix printing, and sequence operations
- `python-more_data_structures` — sets, dictionaries, lambda, map, and data manipulation
- `python-exceptions` — exception handling, safe printing, raising errors, and cleanup logic
- `python-test_driven_development` — doctests, unit tests, documentation, and edge cases
- `python-classes` — classes, objects, private attributes, properties, and linked lists
- `python-more_classes` — rectangles, class attributes, static methods, class methods, and N queens
