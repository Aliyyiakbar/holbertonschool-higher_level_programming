# Python - Classes and Objects

This directory contains my solutions for the Holberton School project
**"Python - Classes and Objects"**.

The goal of this project is to practice object-oriented programming in Python,
including classes, instances, private attributes, properties, setters,
validation, methods, printable objects, and a singly linked list.

## Requirements

- OS: Ubuntu 20.04 LTS
- Interpreter: `python3` (version 3.8.5)
- Style: `pycodestyle` (version 2.7.*)
- All files must be executable
- All files must end with a new line
- The first line of all files must be exactly `#!/usr/bin/python3`
- All modules must have documentation
- All classes must have documentation
- All methods must have documentation
- A `README.md` file at the root of this project folder is mandatory
- File length will be tested using `wc`
- No modules may be imported unless the task allows it

## Files

- `0-square.py`  
  Defines an empty `Square` class.

- `1-square.py`  
  Defines a `Square` class with a private size attribute.

- `2-square.py`  
  Adds validation for the square size.

- `3-square.py`  
  Adds an `area` method that returns the square area.

- `4-square.py`  
  Adds getter and setter methods for the private size attribute.

- `5-square.py`  
  Prints a square using the `#` character.

- `6-square.py`  
  Adds a position attribute to control square printing.

- `100-singly_linked_list.py`  
  Defines `Node` and `SinglyLinkedList` classes with sorted insertion.

- `101-square.py`  
  Makes a `Square` instance printable with the same behavior as `my_print`.

- `102-square.py`  
  Allows square instances to be compared by area.

## Usage

Example:

```bash
chmod +x 0-square.py
./0-main.py
```
